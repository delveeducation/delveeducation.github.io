Python Práctico

Básico • 5 horas

Aprende Python desde cero hasta nivel intermedio con teoría, práctica y un proyecto final aplicado.

¿Qué aprenderás?

Fundamentos de Python y tu primer script.

Funciones, manejo de errores y uso de librerías.

Tipos de datos, operadores y control de flujo.

Programación Orientada a Objetos y técnicas intermedias.

Estructuras de datos: listas, diccionarios, tuplas y sets.

Aplicaciones prácticas y un proyecto final integrador.

Contenido del Curso

Módulo 1: Fundamentos y Primer Script

Instala Python, configura tu entorno, explora Google Colab y crea tu primer programa con variables.

Módulo 2: Tipos de Datos y Control de Flujo

Trabaja con números, cadenas y booleanos, y aprende a usar condicionales y bucles en tus programas.

Módulo 3: Estructuras de Datos

Aprende a usar listas, diccionarios, tuplas y sets para organizar y manipular datos eficientemente.

Módulo 4: Funciones y Manejo de Errores

Define funciones, utiliza parámetros, maneja excepciones y explora funciones lambda.

Módulo 5: Archivos y Librerías

Lee y escribe archivos, utiliza librerías estándar como os y sys, y explora numpy y pandas.

Módulo 6: Programación Orientada a Objetos

Crea clases, objetos y métodos, e implementa herencia simple en Python.

Módulo 7: Python Intermedio

Descubre list comprehensions, generadores con yield y decoradores básicos para funciones.

Módulo 8: Aplicaciones Prácticas

Aplica Python en automatización de tareas, web scraping con BeautifulSoup y gráficos con matplotlib.

Proyecto Final: Gestor de Notas

Integra todo lo aprendido creando un gestor de notas que lea datos desde CSV, procese estadísticas y genere gráficos.


Descripción

Aprenderás desde la instalación y tu primer script, hasta estructuras de datos, funciones y manejo de errores. Explorarás librerías clave como pandas y numpy, además de conceptos de programación orientada a objetos. Descubrirás técnicas intermedias como list comprehensions, generadores y decoradores. Pondrás en práctica lo aprendido con aplicaciones reales de automatización, web scraping y visualización de datos. Ideal para estudiantes y profesionales que quieran pasar de cero a nivel intermedio con ejemplos prácticos.
